/*
COINCIDENCIAS BÁSICAS:
. = Cualquier caracter menos salto de línea
\d = Cualquier dígito (0-9) != \D
\w = Caracter de palabra(a-z, A-Z, 0-9, _) != \D
\s = Espacios de cualquier tipo (espacio, tab, salto de linea) != \S

LIMITES:
\b = Límite de palabra != \B
^ = Inicio de cadena de texto
$ = Final de cadena de texto

CUANTIFICADORES:
* = 0 o más
+ = 1 o más
? = 0 o 1
{#} = Exactamente # veces
{min, max} = Rango

CONJUNTO DE CARACTERES
[] = Caracteres dentro de los brackets != [^]

GRUPOS
() = Grupos
| = uno u otro
*/
